2155 /* \brief Create ghost entities between parts.
2156  *
2157  * Ghost entities are specified similar to 2nd-order adjacencies, i.e.,
2158  * through a "bridge" dimension. The number of layers is measured from
2159  * the inter-part interfaces. For example, to get two layers of region
2160  * entities in the ghost layer, measured from faces on the interface,
2161  * use ghost_dim=3, bridge_dim=2, and num_layers=2.
2162  * The number of layers specified is with respect to the global mesh;
2163  * that is, ghosting may extend beyond a single neighboring processor if the
2164  * number of layers is high.
2165  *
2166  * Ghost information is cached in the partition.
2167  * The triplet describing a ghosting "rule" (ghost dim, bridge dim, #
2168  * layers) is stored in the partition; ghosting that became incorrect
2169  * due to mesh modification or redistribution of mesh entities is
2170  * re-established using these rules by the end
2171  * of iMeshP_syncPartitionAll and iMeshP_syncMeshAll.
2172  * Implementations can choose to keep ghosting consistent throughout
2173  * mesh modification, but ghosts are not required to be consistent until
2174  * the end of these two functions.
2175 
2176  * iMeshP_createGhostEntsAll is cumulative; that is, multiple calls can only
2177  * add more ghosts, not eliminate previous ghosts.
2178  *
2179  * COMMUNICATION: Collective. Blocking.
2180  *
2181  * \param instance (In) Mesh instance containing the partition.
2182  * \param partition (In) The partition in which to create ghosts.
2183  * \param ghost_type (In) Entity type of entities to be ghosted.
2184  * \param bridge_type (In) Entity type through which bridge
2185  * adjacencies are found.
2186  * \param num_layers (In) Number of layers of ghost entities.
2187  * \param include_copies (In) Flag indicating whether to create ghosts
2188  * of non-owned part boundary entities
2189  * (YES=1, NO=0).
2190  * \param err (Out) Error code.
2191  */
2192 void iMeshP_createGhostEntsAll(
2193  iMesh_Instance instance,
2194  iMeshP_PartitionHandle partition,
2195  int ghost_type,
2196  int bridge_type,
2197  int num_layers,
2198  int include_copies,
